# iNotebook

Welcome to iNotebook, your go-to application for managing and organizing your notes seamlessly.

## Features

- **Create Notes**: Easily add new notes with titles, descriptions, and tags.
- **Edit Notes**: Update your notes with the latest information.
- **Delete Notes**: Remove notes you no longer need.
- **Search and Filter**: Quickly find notes using the search feature.
- **Pagination**: Navigate through multiple pages of notes efficiently.

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Sarika-Agrawal/iNotebook.git
   cd iNotebook
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

## Usage

1. **Adding a Note**: Click on "Add Note" and fill in the title, description, and tag.
2. **Editing a Note**: Click the edit icon on the note you want to update, make changes, and save.
3. **Deleting a Note**: Click the delete icon on the note you want to remove.
4. **Searching Notes**: Use the search bar to find notes quickly.
5. **Navigating Pages**: Use the pagination controls to move between pages of notes.

## Technologies Used

- **Frontend**: React, Context API
- **Backend**: Node.js, Express.js
- **Database**: MongoDB
- **Authentication**: JWT
